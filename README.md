# OrderProcessSystem_Async
## make
```
cd OrderProcessSystem_async
make
```
## run server
```
./OPSAsyncServer
```
## run client
```
./OPSAsyncClient
// push new order:
N <new orders request file>
// cancel order:
C <order ID>
// query order:
Q
```
## generate new order requests
```
./Generate
```

## clean
```
make clean
```
